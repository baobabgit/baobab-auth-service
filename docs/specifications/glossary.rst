Glossaire
=========

.. glossary::

   US
      *User Story*. Expression d'un besoin du point de vue de l'utilisateur.

   FEAT
      *Feature*. Fonctionnalité concrète découpant une US.

   Backlog
      Ensemble des tâches (TASK) à réaliser, suivies dans GitHub Issues/Projects.

   Sprint
      Itération de durée fixe (champ *Iteration* du Project) regroupant des tâches.
